# CPS843 Project

# To Run (Instructions Provided Are For Windows Machines):
1. download the CPS843_Project.zip
2. unzip the file
3. cd to file location
4. create a new python venv by running: python -m venv venv
5. activate the venv by running venv\Scripts\activate
6. download the project requirements by running: pip install -r requirements.txt (this may take a few minutes)
7. cd to the python_codes folder
8. run the object detection algorithms by running the different python files (this may take a few minutes the first time you run it)
9. this runs the yolov3u, yolov8n and yolo11n algorithms on the images in the 'images' folder. Currently there are 10 example images.
10. the results will be printed to the terminal, and resulting images saved in seperate folders.
